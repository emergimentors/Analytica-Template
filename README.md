# Analytica Template

Analytica is a minimalist and elegant data portfolio template crafted for showcasing analytical projects and technical expertise.  
It is designed with clarity, performance, and professional presentation in mind.

Perfect for:
- Data Analysts  
- Business Analysts  
- BI Developers  
- Data Science students  
- Professionals building a clean portfolio

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/emergimentors/Analytica-Template.git
```

### 2. Open in your editor
Use VS Code or any code editor.

### 3. Customize content
Update the following:

- Name  
- About section  
- Experience  
- Skills  
- Projects  
- Images  
- Contact info  

### 4. Deploy
You can deploy using:

- GitHub Pages  
- Netlify  
- Vercel  
- Any static hosting  

---

## Customization Guide

### Edit Text Content
Open `index.html` and replace placeholder text with your details.

### Replace Images
Add your images in:
```
assets/images/
```

### Modify Styling
Edit styles in:
```
assets/styles/
```

### Add Projects
Duplicate project cards and update content for your own work.
